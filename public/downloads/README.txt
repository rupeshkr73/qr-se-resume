STEP 1: Download karo (Shop Admin Login se)
─────────────────────────────────────────
✅ QR Code (shop pe lagane ke liye)
✅ Print Agent (print_agent.py - ye file)
✅ Python 3.12 (must install)
✅ SumatraPDF (must install)

STEP 2: Python Install karo
─────────────────────────────────────────
- python.org/downloads pe jao
- Download karo aur install karo
- IMPORTANT: "Add Python to PATH" checkbox ZAROOR tick karo!

STEP 3: SumatraPDF Install karo
─────────────────────────────────────────
- sumatrapdfreader.org pe jao aur install karo
- YA Google mein "SumatraPDF download" search karo
- Agar download slow hai to thoda wait karo ya Google se dhundho

STEP 4: INSTALL.bat Run karo
─────────────────────────────────────────
- INSTALL.bat pe double-click karo
- Ye automatically:
  * EXE file banayega
  * System tray mein add karega
  * Windows startup mein register karega
  * Agent start kar dega

STEP 5: System Tray Check karo
─────────────────────────────────────────
- Taskbar ke neeche right side mein (^) arrow pe click karo
- QR Se Resume icon dikhna chahiye
- Right-click karo options dekhne ke liye

AUTO-UPDATE:
─────────────────────────────────────────
- Agent khud hi update hota rahega
- Koi manual kaam nahi karna
- Jab bhi update aayega, agent khud restart ho jayega

PRINTER:
─────────────────────────────────────────
- Default printer use hoga (jo Windows mein set hai)
- Printer change karne ke liye Windows Settings mein jao
